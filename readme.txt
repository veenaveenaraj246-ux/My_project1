# 🎉 Party Invitation Project  

This is a simple project that creates a **party invitation page** using HTML, CSS, and JavaScript.  
It can be used for birthdays, celebrations, or any special events.  

---

## 🚀 Features
- Attractive invitation design  
- Event details (date, time, venue)  
- RSVP/confirmation option  
- Responsive layout for mobile and desktop  

---

## 🛠️ Technologies Used
- **HTML5** – Structure of the invitation  
- **CSS3** – Styling and animations  
- **JavaScript** – Interactive elements
